HTML & CSS Quick-Reference Guide

HTML Tags:

<h1> to <h6>

Defines different levels of headings.

<h1>This is a Heading</h1>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink.

<a href="https://www.example.com">Visit Example</a>

<div>

Defines a division or section.

<div>This is a division</div>

<marquee>

Defines scrolling text behavior.

<marquee direction="left">Scrolling Text</marquee>

<table>

Defines a table.

<table>
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
        <tr>
            <td>John</td>
            <td>30</td>
        </tr>
    </table>

<sup> and <sub>

Defines superscript and subscript text.

<p>This is a <sup>superscript</sup> and this is a <sub>subscript</sub>.</p>

<input type="checkbox">

Defines a checkbox input.

<input type="checkbox" name="gender" value="male"> Male
    <input type="checkbox" name="gender" value="female"> Female

<input type="file">

Defines a file input.

<input type="file" name="fileUpload">

<input type="submit">

Defines a submit button.

<input type="submit" value="Submit">

<button>

Defines a clickable button.

<button>Click Me</button>

<input type="color">

Defines a color picker input.

<input type="color" name="colorPicker">

<input type="date">

Defines a date input.

<input type="date" name="dateInput">

<input type="time">

Defines a time input.

<input type="time" name="timeInput">

<input type="radio">

Defines a radio button input.

<input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label>

<img>

Defines an image.

<img src="example.jpg" alt="Example Image">

<audio> and <video>

Embeds audio and video content.

<audio controls>
        <source src="example.mp3" type="audio/mpeg">
        Your browser does not support the audio tag.
    </audio>
    
    <video controls>
        <source src="example.mp4" type="video/mp4">
        Your browser does not support the video tag.
    </video>

<style>

Defines style information for a document.

<style>
        p {
            color: blue;
        }
    </style>

<ol> and <li>

Defines an ordered list and list items.

<ol>
            <li>Item 1</li>
            <li>Item 2</li>
        </ol>

<ul> and <li>

Defines an unordered list and list items.

<ul>
            <li>Item 1</li>
            <li>Item 2</li>
        </ul>

<!-- -->

Defines a comment.

<!-- This is a comment -->

HTML class and id

Defines class and id attributes for elements.

<p class="example" id="exampleId">This is a paragraph with class and id attributes.</p>

CSS Properties:

color

Sets the text color.

p {
        color: blue;
    }

font-size

Sets the font size.

h1 {
        font-size: 24px;
    }

text-align

Aligns text.

div {
        text-align: center;
    }

background-color

Sets the background color.

body {
        background-color: #f0f0f0;
    }

margin

Sets the margin around an element.

ul {
        margin: 10px;
    }

padding

Sets the padding inside an element.

li {
        padding: 5px;
    }

border

Sets the border around an element.

table {
        border: 1px solid black;
    }

display

Sets the display behavior of an element.

button {
        display: block;
    }
